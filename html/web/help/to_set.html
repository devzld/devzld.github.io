首次使用如何设置?

首次使用会进入设置向导页面，需要进行如下三项设置：

一、设置通知访问权限

二、设置允许后台运行

三、选择应用加入允许通知白名单

一、设置通知访问权限

第一步：点击通知访问

第二步：开启通知

二、设置允许后台运行

第一步：点击后台运行

第二步：根据手机类型做不同的配置，以允许聚合通知后台运行

点我查看不同手机如何设置

三、选择应用加入允许通知白名单

第一步：点击选择应用

第二步：勾选允许通知的应用名，系统已为您做了一些勾选建议，你只需个性化选择一些即可，
勾选的允许发送通知， 未勾选的应用的通知将被拦截聚合在本软件中显示，新加入的应用默认拦截通知

建议：微信、QQ、企业微信、钉钉等通讯类的软件建议请允许通知， 以免错过及时消息
